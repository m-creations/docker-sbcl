docker-sbcl
===========

An image based on [OpenWrt](http://openwrt.org) x86_64 which runs
[SBCL (Steel Bank Common Lisp)](http://sbcl.org).

How to use
----------

```
docker run -it -v ~/common-lisp:/usr/share/common-lisp/source mcreations/sbcl
```
